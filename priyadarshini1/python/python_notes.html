Python Basics

1.1 Introduction: Features, History and Applications

What is Python? Python is a high-level, interpreted, general-purpose programming language. It is easy to learn and widely used for developing applications, automation, data analysis, artificial intelligence, and embedded software.

Features of Python:

  • Simple and Easy to Learn: Uses simple English-like syntax, making it beginner-friendly.
  • Interpreted Language: Code is executed line by line, which makes debugging easy.
  • High-Level Language: Programmer does not need to manage memory manually.
  • Platform Independent: Programs can run on Windows, Linux, and macOS without changes.
  • Free and Open Source: Python is freely available and can be modified.
  • Large Standard Library: Provides built-in modules for math, file handling, networking, etc.
  • Object-Oriented: Supports concepts like classes and objects.

Applications of Python:

  1. Web Development (Django, Flask)
  2. Data Analysis, AI, and ML applications
  3. Automation and Scripting
  4. Scientific and Engineering Applications
  5. Embedded Systems Support (MicroPython)
  6. Game Development (Pygame)
  7. Desktop Applications (GUI)

1.2 Python Building Blocks

Indentation

Indentation refers to the spaces at the beginning of a line of code. Python uses indentation to define blocks of code (if/else, loops, functions). Generally, 4 spaces are used for indentation.

if x > 0:
    print("Positive number")

Identifiers

Names given to variables, functions, classes, etc.

Rules for Identifiers:

  • Must start with a letter (a-z or A-Z) or underscore (_)
  • Case-sensitive (age and Age are different)
  • Cannot start with a digit
  • Cannot use Python keywords

Examples: Valid: total, _count, marks1 | Invalid: 1num, total-marks, if

Variables

A variable is a name that refers to a memory location used to store data. Python is dynamically typed.

x = 10
name = "Python"

Comments

Comments are used to explain code and are ignored by the Python interpreter.

# This is a single-line comment

"""
This is a
multi-line comment
"""

Keywords

Reserved words with special meaning in Python (if, else, for, while, etc.). Cannot be used as identifiers.

import keyword
print(keyword.kwlist)

1.5 Control Flow Statements

Control flow statements are used to control the execution flow of a Python program.

Conditional Statements

if Statement:

x = 10
if x > 0:
    print("Positive number")

if-else Statement:

x = -2
if x > 0:
    print("Positive")
else:
    print("Negative")

if-elif-else Statement:

marks = 75
if marks >= 75:
    print("Distinction")
elif marks >= 60:
    print("First Class")
else:
    print("Pass")

Nested if Statement:

x = 10
if x > 0:
    if x % 2 == 0:
        print("Positive Even Number")
    else:
        print("Positive Odd Number")

Loops in Python

while Loop:

i = 1
while i <= 5:
    print(i)
    i += 1

for Loop:

for i in range(1, 6):
    print(i)

Nested Loops:

for i in range(1, 3):
    for j in range(1, 3):
        print(i, j)

Loop Manipulation Statements

break Statement: Terminates the loop immediately.

for i in range(1, 6):
    if i == 3:
        break
    print(i)

continue Statement: Skips the current iteration and continues with the next.

for i in range(1, 6):
    if i == 3:
        continue
    print(i)

pass Statement: Null statement that does nothing (used as a placeholder).

for i in range(5):
    pass # Placeholder for future code

else with Loop: Executes when the loop completes normally (without break).

for i in range(1, 4):
    print(i)
else:
    print("Loop completed")

Data Structures in Python

2.1 Lists

A list in Python is a collection of items stored in a single variable. Lists are ordered, mutable (can be changed), and allow duplicate values.

Defining and Accessing Lists

# Defining lists
numbers = [10, 20, 30]
names = ["Amit", "Ravi", "Neha"]
mixed = [1, "Python", 3.5]

# Accessing values
print(numbers[0]) # 10
print(numbers[-1]) # 30 (negative indexing)

Updating and Deleting List Values

# Updating list values
numbers = [10, 20, 30]
numbers[1] = 50
print(numbers) # [10, 50, 30]

# Deleting list values
del numbers[1] # deletes element at index 1
numbers.remove(30) # deletes value 30
numbers.pop() # deletes last element

Basic List Operations

# Concatenation
list1 = [1, 2]
list2 = [3, 4]
print(list1 + list2) # [1, 2, 3, 4]

# Repetition
print([1, 2] * 3) # [1, 2, 1, 2, 1, 2]

# Membership
list1 = [10, 20, 30]
print(20 in list1) # True

# Length
print(len(list1)) # 3

# Slicing
numbers = [10, 20, 30, 40, 50]
print(numbers[1:4]) # [20, 30, 40]

Built-in List Methods

2.2 Tuples

A tuple is similar to a list but immutable (cannot be changed). Tuples are ordered and allow duplicate values.

2.3 Sets

A set is an unordered, mutable collection of unique elements. Sets are mainly used to remove duplicates and to perform mathematical set operations.

Defining Sets

# Defining sets
s = {10, 20, 30, 40}
print(s)

# Using set() constructor
s = set([10, 20, 30, 40])

# Empty set
s = set() # Not {}

Accessing and Deleting Set Values

# Accessing values (sets are unordered)
s = {10, 20, 30}
for value in s:
    print(value)

# Deleting set values
s.remove(20) # removes specific element
s.pop() # removes random element
del s # deletes entire set

Set Operations

A = {1, 2, 3}
B = {3, 4, 5}

# Union (all elements from both sets)
print(A | B) # {1, 2, 3, 4, 5}

# Intersection (common elements)
print(A & B) # {3}

# Difference (elements in A but not in B)
print(A - B) # {1, 2}

# Symmetric Difference (elements in either set but not both)
print(A ^ B) # {1, 2, 4, 5}

Set Methods

2.4 Dictionaries

A dictionary is a collection of key-value pairs. Keys are unique and immutable, while values can be of any data type.

Defining and Accessing Dictionaries

# Defining dictionary
student = {
    "name": "Rahul",
    "roll": 101,
    "marks": 85
}

# Accessing values
print(student["name"]) # Rahul
print(student.get("marks")) # 85

Updating and Deleting Dictionary Values

# Updating values
student["marks"] = 90 # change existing value
student["grade"] = "A" # add new key-value pair

# Deleting values
del student["roll"] # deletes key "roll"
student.pop("marks") # removes key "marks"
student.popitem() # removes last inserted item

Dictionary Operations

student = {"name": "Rahul", "roll": 101, "marks": 85}

# Traversing a dictionary
for key in student:
    print(key, student[key])

# Check key existence
if "name" in student:
    print("Key exists")

# Length of dictionary
print(len(student)) # 3

# Copy dictionary
new_student = student.copy()

Object Oriented Programming in Python

4.1 Object Oriented Concepts

A class in Python is a user-defined template for creating objects. It bundles data and functions together.

Creating Class and Object

# Simple class
class Student:
    pass # empty class

# Creating object
s1 = Student()

# Class with attributes
class Bike:
    name = ""
    gear = 0

# Create object and access attributes
bike1 = Bike()
bike1.gear = 4
bike1.name = "Splendor"
print(f"Name: {bike1.name}, Gear: {bike1.gear}") # Name: Splendor, Gear: 4

4.2 Constructors in Python

A constructor is a special method that runs automatically when an object is created. In Python, it's defined using __init__().

Non-Parameterized Constructor

class Student:
    def __init__(self):
        print("Student object created")

s1 = Student() # Output: Student object created

Parameterized Constructor

class Student:
    def __init__(self, name, roll):
        self.name = name
        self.roll = roll

s1 = Student("Amit", 101)
print(s1.name) # Amit
print(s1.roll) # 101

4.3 Polymorphism

Polymorphism allows methods to do different things based on the object it is acting upon.

Method Overloading

Defining multiple methods with the same name but different parameters. Note: Python doesn't support traditional method overloading.

def product(a, b):
    p = a * b
    print(p)

def product(a, b, c):
    p = a * b * c
    print(p)

# Only the last definition is recognized
product(4, 5, 5) # 100
# product(4, 5) would raise an error

Method Overriding

Writing a method in the child class with the same name and parameters as a method in the parent class.

# Parent class
class Mobile:
    def call(self):
        print("Mobile calling")

# Child class
class Samsung(Mobile):
    def call(self):
        print("Samsung calling")

# Object creation
m = Mobile()
s = Samsung()

m.call() # Mobile calling
s.call() # Samsung calling (overridden method)

4.4 Data Hiding / Abstraction

Data hiding is achieved by making class members private using double underscore prefix (__).

class BankAccount:
    def __init__(self):
        self.__balance = 0 # private attribute

    def deposit(self, amount):
        self.__balance += amount

    def get_balance(self):
        return self.__balance

account = BankAccount()
account.deposit(1000)
print(account.get_balance()) # 1000
# account.__balance would cause error (private attribute)

4.5 Inheritance

Inheritance allows a class to inherit attributes and methods from another class.

Single Inheritance

# Parent class
class Animal:
    def speak(self):
        print("Animal speaks")

# Child class
class Dog(Animal):
    def bark(self):
        print("Dog barks")

d = Dog()
d.speak() # Animal speaks (inherited)
d.bark() # Dog barks

Multiple Inheritance

# Parent class 1
class Father:
    def height(self):
        print("Tall")

# Parent class 2
class Mother:
    def color(self):
        print("Fair")

# Child class inheriting from both
class Child(Father, Mother):
    pass

c = Child()
c.height() # Tall
c.color() # Fair

Multilevel Inheritance

# Grandparent class
class GrandParent:
    def property(self):
        print("Land")

# Parent class
class Parent(GrandParent):
    def house(self):
        print("Bungalow")

# Child class
class Child(Parent):
    def car(self):
        print("BMW")

c = Child()
c.property() # Land
c.house() # Bungalow
c.car() # BMW
